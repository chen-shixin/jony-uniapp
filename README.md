# Jony-uniapp

## 一拍即传测试 APP

### 1.运行

	1.拷贝跟目录下debug文件夹到根目录unpackage文件夹下 （.apk文件为安卓自定义基座）
	
	2.菜单运行——运行到手机或模拟器——运行基座选择——自定义Android基座
	
	3.选择项目根目录直接运行在IOS模拟器或链接Android手机运行
